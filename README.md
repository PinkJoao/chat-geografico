# chat-geografico
Aplicação de chat desenvolvida em python durante a cadeira de programação paralela e distribuída, que permite a comunicação com síncrona e assíncrona se baseando na posição geográfica do usuário

foi utilizado neste projeto a biblioteca linsimpy (https://github.com/robwalton/linsimpy) para a implementação do espaço de tuplas compartilhado
